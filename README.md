# Smart Room Automation System

This is an Arduino-based embedded systems project designed to automate
room lighting, temperature control, and gas safety using multiple sensors.

## Features
- Automatic and Manual mode switching
- PIR-based human motion detection
- LDR-based smart lighting control
- Temperature-controlled fan using PWM
- Gas leakage detection with emergency alert
- 16x2 I2C LCD for real-time system status

## Components Used
- Arduino UNO
- PIR Sensor
- LDR
- TMP36 Temperature Sensor
- MQ-2 Gas Sensor
- DC Motor (Fan)
- LED
- Buzzer
- Push Button
- 16x2 I2C LCD

## Tools & Technologies
- Arduino IDE
- Tinkercad
- Embedded C
